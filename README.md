## 🚀 Proyecto destacado — Tesis

<div align="center">

> **Plataforma web de asistencia visual con IA**
> 
> Aplicación que ayuda a personas con discapacidad visual a interpretar su entorno en tiempo real mediante visión computacional e inteligencia artificial generativa. Retroalimentación de audio, cumplimiento WCAG 2.1 y arquitectura escalable.

| Tecnología | Rol |
|---|---|
| Google Vision API + OpenCV | Análisis visual en tiempo real |
| GPT-4o | Descripción contextual inteligente |
| Vue.js | Interfaz accesible WCAG 2.1 |
| Python | Backend de procesamiento |
| SCRUM | Metodología de desarrollo |

</div>
